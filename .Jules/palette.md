## 2026-06-01 - Custom CSS Constraints
**Learning:** Adding custom CSS rules for specific UX enhancements in `App.css` may violate explicit design system boundaries or user constraints, even if it aims to resolve documented UX issues.
**Action:** When implementing UX improvements, strictly adhere to modifying JSX with existing classes or robust accessibility attributes (e.g. `aria-label`, `role`, `tabIndex`) to avoid structural styling regressions.
